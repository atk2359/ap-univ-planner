# 愛知県立大学大学院 修士課程 履修シミュレーター (Academic Planner)

本ツールは、愛知県立大学大学院 修士課程における講義履修計画の策定を支援するための非公式Webアプリケーションです。
複雑な修了要件（専攻間振替や単位区分）を自動で計算し、2年間の履修計画を視覚的に管理できます。

## 🚀 公開URL
`https://atk2359.github.io/ap-univ-planner/`

## ✨ 愛知県立大学大学院向けの機能
- **修了要件の自動判定**: 専門科目、専攻科目、関連科目、演習科目、特別研究の各区分に基づいた単位集計。
- **他専攻科目の振替計算**: 専攻科目への振替（上限4単位）や、関連科目への振替（上限2単位）を自動計算。
- **学期別時間割**: 愛知県立大学の学期区分（前期・後期）に合わせ、M1〜M2の計4学期を管理。
- **集中講義・特別研究対応**: 集中講義や特別研究単位も専用枠で管理可能。

## 🛠 使い方
1. **所属コース（専攻）の選択**: 左上のメニューから自分の所属を選択してください。
2. **講義の登録**: カタログから講義名を選択し、開講学期・時限を指定して時間割に追加します。
3. **要件チェック**: 右パネルの「修了要件 進捗」を確認し、不足単位がないかチェックします。
4. **データの保存**: 右上の「保存(.json)」から、作成した計画をファイルとして保存できます。

## 📂 ファイル構成
- `index.html`: アプリケーションのUI構造。
- `script.js`: 時間割制御、要件計算、JSON入出力等のメインロジック。
- `data.js`: 標準の講義データセット（共通科目・各専攻科目）。
- `style.css`: 視認性を重視したUIデザイン。
- `diff_template.json`: カタログ拡張用の配布用テンプレート。

## 🔧 開発者向け：データの更新方法

ベースとなる講義データを更新する場合、`data.js` 内の `coreCourses` および `majorMasters` を直接編集してください。
編集後、`lastUpdated` 変数に日付を記述することで、画面上の「Data: YYYY-MM-DD」表示が更新されます。
